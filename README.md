🔐 Windows Password Manager (C++ | WinAPI)
 
A GUI-based Password Manager built using C++ and the Windows API (Win32).
This application allows users to securely store, search, and generate passwords using basic encryption techniques.

🚀 Features

🔒 XOR-based encryption with HEX encoding

💾 Secure credential storage in file

🔍 Search stored credentials

🔑 Random strong password generator

📊 Password strength checker

🖥️ Desktop GUI built using WinAPI

🛠 Technologies Used

C++

Windows API (Win32)

File Handling (C-style FILE operations)

XOR Cipher Encryption

Hex Encoding / Decoding

🧠 How It Works

User enters:

Website

Username

Password

Data is:

Encrypted using XOR cipher

Converted into HEX format

Stored inside passwords.txt

When searching:

Data is decrypted

Original credentials are displayed
